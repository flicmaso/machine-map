# GBP Packaging SCADA Proof of Concept

Static GitHub Pages demo for a plant-owned condition monitoring and SCADA/historian concept.

## What It Demonstrates

- Live simulated vibration and bearing temperature for the North and South baler rooms.
- 13 blower-room monitoring points plus Bobst die-cutter electrical pilot assets.
- Ignition-style plant overview, alarm states, detail drill-downs, trend charts, and active events.
- Recommended plant-owned architecture: open-protocol sensors and gateways into Ignition/UNS, SQL or Canary historian, Power BI only for summaries.
- Staged rollout plan and vendor gate questions based on open data egress.

Open `index.html` directly or serve this folder with GitHub Pages.
