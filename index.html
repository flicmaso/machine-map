NFC tags — how this connects

Each cutter gets one NFC sticker on the operator-side frame. The tag stores just a URL — tapping a phone opens this map already on that machine:

Deep links: ?cutter=12 picks the machine, &sec=platen zooms to a section, &walk=1 opens the weekly walkdown immediately. Write the URL to NTAG 215 stickers (~$0.30 each) with any free NFC app — iPhone and Android read them natively.

Tap tag → map opens on that cutter → log entry or run walkdown. No hunting for the right spreadsheet at the machine.
